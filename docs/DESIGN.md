## Overview

Halo's documentation site uses a **warm cream editorial canvas** with **Indigo accent**. The base atmosphere is a tinted cream canvas (`#faf9f5`) — distinctly warm, deliberately not the cool gray-white of typical developer docs. Headlines run a serif display (Cormorant Garamond) paired with Noto Sans SC / Inter body sans.

Brand voltage comes from the **cream + indigo** pairing. Indigo (`#6366f1`) is the signature Halo accent, used on CTAs, code highlights, and link states. The warm editorial feel signals that Halo is a thoughtful, well-crafted tool — not just another npm package.

Three surface modes alternate:
1. **Cream canvas** (`#faf9f5`) — default page background
2. **Light cream cards** (`#efe9de`) — feature cards, content containers
3. **Dark navy surfaces** (`#181715`) — code windows, comparison sections, CTAs, footer
