# 教程

通过分步教程学习使用 Halo 构建真实应用。

## 可用教程

- [构建一个聊天机器人](/zh/tutorial/build-a-chatbot) — 从零开始构建带工具调用、流式输出和缓存优化的 AI 聊天机器人的完整指南。

## 前置条件

开始教程之前，请确保：

- 已安装 **Node.js 18+**
- 拥有 **DeepSeek API 密钥**（或其他受支持的提供商）
- 基本熟悉 **TypeScript** 和 **async/await**
- 已完成 [5 分钟上手](/zh/getting-started/quick-start) 指南

## 你将学到

| 教程 | 涵盖概念 | 时间 |
|---|---|---|
| 构建聊天机器人 | Agent 创建、工具调用、流式输出、缓存监控 | ~30 分钟 |
