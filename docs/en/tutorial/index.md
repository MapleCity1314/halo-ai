# Tutorial

Step-by-step tutorials to build real applications with Halo.

## Available Tutorials

- [Build a Chatbot](/en/tutorial/build-a-chatbot) — A complete step-by-step guide to building an AI chatbot with tool calling, streaming, and cache optimization.

## Prerequisites

Before starting the tutorials, make sure you have:

- **Node.js 18+** installed
- A **DeepSeek API key** (or other supported provider)
- Basic familiarity with **TypeScript** and **async/await**
- Completed the [Quick Start](/en/getting-started/quick-start) guide

## What You'll Learn

| Tutorial | Concepts Covered | Time |
|---|---|---|
| Build a Chatbot | Agent creation, tool calling, streaming, cache monitoring | ~30 min |
